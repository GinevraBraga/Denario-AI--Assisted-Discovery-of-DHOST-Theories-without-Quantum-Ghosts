Start from the following quantum corrected action

\begin{multline}
\mathcal{L}_{\text{eff}} = c_0\bar{R} + c_1 \left[\phi_{\mu\nu}\phi^{\mu\nu} - (\Box\phi)^2\right] + A_4(\phi, X) C_{\mu\nu\rho\sigma} \phi^{\mu\nu}\phi^{\rho\sigma} + A_5(\phi, X) G_{\mu\nu}\phi^\mu\phi^\nu \\
+ \beta_{\text{GB}} \left(\bar{R}^2 - 4\bar{R}_{\mu\nu}^2 + \bar{R}_{\mu\nu\rho\sigma}^2\right) + \beta_{W^2} C_{\mu\nu\rho\sigma}^2
\end{multline}
with $\beta_{GB},\beta_{W^2},c_1$ constants and 
\begin{align}
A_2 &= -c_1 \,,\\
A_4 &= \frac{-16\bar{X}c_1^3 + 12c_0c_1^2}{8(c_0 - \bar{X}c_1)^2}\,, \\
A_5 &= \frac{4c_1^3}{8(c_0 - \bar{X}c_1)^2}\,.
\end{align}

Check explicitly whether this action remains invariant under the following transformations
$$\delta_\epsilon \phi(x) = \epsilon(x) \Lambda(\phi, X)$$
and 
$$\delta_\epsilon g_{\mu\nu}(x) = \epsilon(x) \mathcal{L}_{\xi} g_{\mu\nu}$$
where $\mathcal{L}_{\xi}$ denotes the Lie derivative along a vector field $\xi^\mu$. This vector field is constructed from the scalar field itself, $\xi^\mu = \alpha(\phi, X) \phi^\mu$, with $\phi^\mu = g^{\mu\nu}\partial_\nu\phi$. Here, $\epsilon(x)$ is an arbitrary spacetime-dependent function, acting as the gauge parameter, while $\Lambda(\phi, X)$ and $\alpha(\phi, X)$ are functions of the scalar field and its kinetic term.
